# Fundamentals 

Each video is composed of scenes, where you describe what appears, how it transitions, and how long it stays.

## Basic Structure

A scene is defined using `## !!scene` followed by a unique name:

```markdown
## !!scene welcome
!duration 5s
!transition enter:fade

Your content here
```

{/* `duration`
: Required ✅
: Scene duration
: Example: `!duration 5s`

`transition`
: Optional ❌ 
: Entry animation
: Default: none
: Example: `!transition enter:fade` */}

- `duration` ✅
  Scene duration
  > `!duration 5s`

- `transition` ❌
  Entry animation (Default: none)
  > `!transition enter:fade`

{/* ## Properties

| Property       | Required | Description             | Default | Example                     |
|---------------|----------|-------------------------|---------|----------------------------|
| `duration`    | ✅       | Scene duration          | -       | `!duration 5s`             |
| `transition`  | ❌       | Entry animation         | none    | `!transition enter:fade`   |
| `font`   | ❌       | Text styling           | -       | `!font fontSize:24px` |
| `layoutUtils` | ❌       | Layout control         | -       | `!layoutUtils padding:20px`|
| `mediaUtils`  | ❌       | Media handling         | -       | `!mediaUtils size:400px`   |
| `codeUtils`   | ❌       | Code block styling     | -       | `!codeUtils theme:dark`    | */}

## Example

```markdown
## !!scene complex
!duration 4s
!transition enter:fade
!font fontSize:24px
!layoutUtils padding:20px
!codeUtils theme:dark
```

## Steps

Steps are the building blocks of your video. Each step represents a scene or frame in the final video.

### Basic Syntax

```markdown
## !!steps 1
!duration 5s
!transition name:fade duration:2s

Your content here
```

### Properties

<Callout>

This is a default callout. You can embed **Markdown** inside a `callout`.

</Callout>

- `!!steps n`: Defines step number (required)
- `!duration`: Length of step in video (required)
  - Format: `Ns` where N is seconds
  - Example: `!duration 5s`
- `!transition`: Animation between steps (optional)
  - Properties: 
    - name: fade, slide-from-top, magic
    - duration: transition length
    - delay: wait before transition
  - Example: `!transition name:fade duration:3s delay:500ms`

### Examples

```markdown
## !!steps 1
!duration 6s
!transition name:fade duration:3s

First scene content

## !!steps 2
!duration 4.5s
!transition name:slide-from-top duration:3s delay:500ms

Second scene content

## !!steps 3
!duration 4.5s
!transition name:magic

Final scene content
```

### Tips

- Number steps sequentially (1, 2, 3...)
- Each step must have a duration
- Keep transitions consistent for better flow
